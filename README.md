# Employee Management System

A Java console application developed to practice Object-Oriented Programming (OOP) concepts.

## Features
- Add Employee
- View Employee Details
- Update Department
- Show Employee Count

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism
- Constructor Overloading
- Method Overloading
- Method Overriding
- Static Variables and Methods

## Technologies
- Java
- VS Code
- Git & GitHub
